# number-search

Number Search Program in C++
Description
This C++ program implements a simple linear search algorithm to find a specified number within an array. The user is prompted to enter a number, and the program checks if that number exists in a predefined list of integers. It then provides feedback on whether the number was found.

Features
Linear Search: The program utilizes a straightforward linear search method to locate the number in the array.
User Input: Prompts the user to input a number for searching.
Clear Output: Displays informative messages indicating whether the number was found in the array.
